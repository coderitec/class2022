the command line commands

  • change directory to desktop: cd desktop
  • to view files on current directory: ls
  • to create a directory: mkdir foldername
  • to create a file inside a folder: touch nameoffile
  • to open current directory with vsc: code .

working with git

  • download git and install: download git
  • navigate to github and create an account
  • go back to your vsc and start working
  • open your terminal on vsc: CTRL + SHIFT + `
  • change your terminal to git bash
  • initialise git repository: git init
  • check git status: git status
  • move out from master branch into a new branch: git checkout -b nameofbranch
  • add the file into the current branch: git add .
  • give a message to your changes using: git commit -m "type your commit message"
  • confirm your git commits using git log

working with github

  1. sign into your github account
  2. create a remote repository
  3. add the remote to origin: git remote add origin repo address
  4. verify that the remote address exist: git remote -v
  5. push the files to the remote repo: git push
  6. copy the command to set upstream if not existing
  7. when changes are made, add and commit the changes
  8. then use git push to update the remote repo
